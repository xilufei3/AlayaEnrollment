# Script 目录脚本运行说明

本文档说明 `D:\AlayaEnrollment\script` 目录下当前可运行脚本的用途和推荐运行指令。

## 运行前提

建议先切到仓库根目录再执行：

```powershell
cd D:\AlayaEnrollment
```

Milvus 和 ETL 服务需要提前可用，常见环境变量来源于项目根目录下的 `.env`。

## 1. 导入单个文件

脚本：`script/ingest_file.py`

推荐运行方式：

```powershell
python -m script.ingest_file --file ".\data\raw\unstructured\本科专业.md" --category major
```

带检索验证：

```powershell
python -m script.ingest_file --file ".\data\raw\unstructured\本科专业.md" --category major --query "本科专业" --top-k 3
```

可选参数：

- `--file`：待导入文件路径，必填
- `--category`：分类，可选值为 `school_info`、`admissions`、`major`、`career`、`campus`
- `--chunk-size`：分块大小，默认 `512`
- `--chunk-overlap`：分块重叠大小，默认 `64`
- `--query`：导入完成后执行一次检索验证
- `--top-k`：检索返回条数，默认 `3`

查看帮助：

```powershell
python -m script.ingest_file --help
```

## 2. 批量导入目录下全部文件

脚本：`script/ingest_all.py`

说明：该脚本在批量导入前，会先清空当前 Milvus collection，再重新导入目录下所有支持文件。

默认导入目录：

```text
D:\AlayaEnrollment\data\raw\unstructured
```

直接使用默认目录：

```powershell
python -m script.ingest_all --category school_info
```

指定目录批量导入：

```powershell
python -m script.ingest_all --dir ".\data\raw\unstructured" --category school_info
```

带检索验证：

```powershell
python -m script.ingest_all --dir ".\data\raw\unstructured" --category school_info --query "南方科技大学" --top-k 3
```

可选参数：

- `--dir`：待递归导入目录，默认 `data/raw/unstructured`
- `--category`：分类，可选值为 `school_info`、`admissions`、`major`、`career`、`campus`
- `--chunk-size`：分块大小，默认 `512`
- `--chunk-overlap`：分块重叠大小，默认 `64`
- `--query`：导入完成后执行一次检索验证
- `--top-k`：检索返回条数，默认 `3`

查看帮助：

```powershell
python -m script.ingest_all --help
```

## 3. 兼容入口

脚本：`script/ingest_vectors.py`

说明：这是批量导入的兼容入口，内部直接转到 `script.ingest_all`。

运行方式：

```powershell
python -m script.ingest_vectors --dir ".\data\raw\unstructured" --category school_info
```

## 4. 推荐使用顺序

如果只是验证链路，优先使用单文件导入：

```powershell
python -m script.ingest_file --file ".\data\raw\unstructured\本科专业.md" --category major --query "本科专业" --top-k 3
```

如果要完整重建当前向量库，使用批量导入：

```powershell
python -m script.ingest_all --dir ".\data\raw\unstructured" --category school_info
```
